## 使用方法
src/app.vue内にあるFirebase Config内を置き換えれば動きます。
なお、使っているFirebaseの機能はFirestore・Hosting・CloudStorage・Authencationですので、コンソールで使えるようにしておいてください。
また、eslintはWindows用です。
